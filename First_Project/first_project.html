Creating Your First .NET Console Application

1. Check .NET SDK Installation

Before you begin, ensure you have the .NET SDK installed. Open your terminal or command prompt and run:

dotnet --version

If a version number is displayed, you're good to go! If not, download and install the .NET SDK from the official website: https://dotnet.microsoft.com/download

2. Create a New Directory

Create a new folder for your project. You can do this using the command line or your file explorer. For example, to create a folder named "HelloWorldProject":

mkdir HelloWorldProject

3. Navigate to the Directory

Use the cd command to navigate into your newly created directory:

cd HelloWorldProject

4. Create the Project

Generate a new console application project using this command:

dotnet new console

5. Run the Application

Finally, run your application:

dotnet run

You should see "Hello World!" printed to your console.
